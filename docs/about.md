# 关于

## 关于这个知识库

这是我的个人知识管理系统，基于 [VitePress](https://vitepress.dev/) 构建，部署在 [GitHub Pages](https://pages.github.com/) 上。

## 知识库结构

```
知识库
├── 工作笔记
│   ├── 项目文档
│   ├── 会议记录
│   └── 技术方案
├── 学习笔记
│   ├── 前端技术
│   ├── 后端技术
│   └── 读书笔记
└── 随手记
    ├── 灵感收集
    ├── 问题记录
    └── 工具推荐
```

## 使用工具

- **写作**：语雀 / VS Code
- **构建**：VitePress
- **部署**：GitHub Pages
- **版本控制**：Git
