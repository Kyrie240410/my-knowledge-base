## 灵感清单

### 2024-01

#### 产品创意：个人知识管理助手

**灵感来源**：
整理笔记时发现，很多知识散落在不同平台，难以统一管理。

**想法描述**：
开发一个浏览器插件，可以：
1. 一键收藏网页内容
2. 自动提取关键信息
3. 支持标签和分类
4. 导出为 Markdown
5. 同步到 GitHub

**可行性分析**：
- ✅ 技术可行
- ✅ 有实际需求
- ⚠️ 需要设计好数据格式
- ⚠️ 考虑隐私问题

**下一步行动**：
- [ ] 调研现有工具（Notion Web Clipper、印象笔记剪藏等）
- [ ] 设计最小可行产品（MVP）
- [ ] 确定技术方案

---

#### 技术想法：自动化文档生成

**灵感来源**：
写文档太耗时，而且容易过时。

**想法描述**：
基于代码注释自动生成文档：
1. 解析代码中的 JSDoc/注释
2. 提取 API 信息
3. 生成 Markdown 文档
4. 支持自定义模板

**参考工具**：
- [JSDoc](https://jsdoc.app/)
- [TypeDoc](https://typedoc.org/)
- [ESDoc](https://esdoc.org/)
