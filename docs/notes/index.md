# 随手记

这里记录日常的灵感、问题和工具推荐。

## 分类

- [灵感收集](./ideas/) - 突如其来的想法
- [问题记录](./issues/) - 遇到的问题和解决方案
- [工具推荐](./tools/) - 好用的工具和服务

## 记录原则

1. **快速记录**：想到就记，不要等
2. **定期整理**：每周回顾，分类归档
3. **建立连接**：相关想法要关联
4. **及时归档**：成熟的想法移到对应分类
