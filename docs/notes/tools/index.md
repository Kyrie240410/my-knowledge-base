# 工具推荐

## 开发工具

### 编辑器

#### VS Code

**推荐理由**：
- 免费开源
- 插件丰富
- 跨平台
- 集成终端

**必备插件**：
| 插件 | 用途 |
|------|------|
| Chinese (Simplified) | 中文语言包 |
| ESLint | 代码检查 |
| Prettier | 代码格式化 |
| GitLens | Git 增强 |
| Markdown All in One | Markdown 支持 |
| Vim | Vim 键位 |

**配置要点**：
```json
{
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.tabSize": 2,
  "files.autoSave": "afterDelay"
}
```

### 效率工具

#### 终端工具

**iTerm2** (Mac)
- 分屏功能
- 搜索功能
- 自动补全

**Windows Terminal** (Windows)
- 多标签页
- 支持 WSL
- 自定义主题

#### 版本控制

**Git 常用命令**：
```bash
# 初始化仓库
git init

# 添加文件
git add .

# 提交更改
git commit -m "提交信息"

# 推送到远程
git push origin main

# 拉取更新
git pull origin main

# 查看状态
git status

# 查看日志
git log --oneline
```
