# 个人知识库 Web 搭建项目

## 项目概述

为用户搭建一套完整的、可随时随地访问的个人知识库 Web 应用，基于 VitePress + GitHub Pages，零成本部署。

## 技术方案

- **静态生成器**: VitePress (Vue 团队出品，极速构建)
- **部署托管**: GitHub Pages (免费 CDN + HTTPS)
- **自动化**: GitHub Actions (推送即部署)
- **写作**: Markdown / 语雀导出

## 核心功能

1. ✅ **全文搜索** - 基于本地搜索，快速定位内容
2. ✅ **响应式设计** - 手机/平板/电脑完美适配
3. ✅ **暗黑模式** - 支持亮色/暗色主题切换
4. ✅ **自动部署** - Git push 后自动构建部署
5. ✅ **Markdown 原生支持** - 专注内容写作
6. ✅ **代码高亮** - 支持多种编程语言
7. ✅ **目录导航** - 自动生成侧边栏

## 使用方式

### 本地开发
```bash
npm install
npm run docs:dev
```

### 构建
```bash
npm run docs:build
```

### 部署
```bash
git push origin main  # 自动触发 GitHub Actions 部署
```

## 访问地址

部署完成后：
```
https://你的用户名.github.io/my-knowledge-base
```

## 下一步行动

1. 用户创建 GitHub 仓库
2. 推送代码到仓库
3. 配置 GitHub Pages
4. 修改 config.mjs 中的个人信息
5. 开始写作
